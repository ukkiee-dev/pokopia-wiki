Building

Creating your own area is a fundamental element of Pokémon Pokopia and beyond just a Pokémon's habitat, you can do a lot of building using various tiles and materials to your own desires. As long as there's four walls and a door, something can be a home for a Pokémon.

This allows for you to create a house for any Pokémon and even create habitats within buildings so long as it has four walls and a door. There is a limit on the size of a structure for it to be considered as a place for a Pokémon. If you exceed an 11x12 size structure on width versus depth then it will stop being classed as a habitat. No matter the size of the self built habitats, you will be able to house four Pokémon within, but Pokémon will not move in unless there's 3 unique pieces of furniture.

Building Kits

However, beyond that there are several kits that you can buy in the store in order to build either an entire dwelling, which can be an internal one like a house which is much larger for decorating, or a bespoke building that can be a home for a Pokémon but doesn't have much space. There are also various decorations that can be built, as well as buildings, machines and even entire concepts which will be filled with rare items such as Stereos.

Each building has a certain size that needs to be taken, and the space that you need to build it can be increased by 1 in order to give room to build.

In addition to that, you can purchase kits to help move or destroy a previously built kit

If you use Tinkaton for building the kits, it reduces the time significantly and a Next Day build becomes just 1 hour.

In each area, there's a tally of 40 building "points" that you can have. Each build takes either 1 or 2 points, depending on their size, towards this tally and once it hits 40, you will be unable to build any more building kits in the area.
